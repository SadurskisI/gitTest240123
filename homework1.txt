1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

на гитхабе в нужном репозитории заходим во вкладку “Code”, где копируем SSH ссылку для дальнейшего клонирования репозитория в свой каталог
в GitBash переходим в папку, в которой будет хранится репозиторий.
выполняем команду “git clone “+ скопированная ранее ссылка на репозиторий,
далее необходимо перейти в папку созданного локально репозитория

2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

“git init” для инициализации локального репозитория,
создаем репозиторий на гитхабе, копируем SSH ссылку
“git remote add origin” и скопированная выше ссылка 
“git add “ и имя файла, изменения в котором нужно сохранить в удаленный репозиторий или “.”, если это все файлы
“git commit -m” “комментарий” и имя файла (либо гит сам закоммитит все добавленные в индекс файлы)
“git push”

3. Как получить на локальный компьютер изменения с github.com
 в GitBash в папке репозитория команда “git pull”

4. Какую информацию нам дают команда :
git status -  С ее помощью можно проверить индексацию изменений и увидеть файлы, которые не отслеживаются Git.
git log - выводит список commit'ов, созданных в данном репозитории в обратном хронологическом порядке.
git remote -v - показывает, с каким удаленным репозиторием связан данный локальный репозиторий (origin fetch и push)
