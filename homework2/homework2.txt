1.Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?
git checkout "file name"

2.Как вернуть файл из индекса в рабочий каталог?
git restore --staged "file name"

3.Как отменить ошибочный commit ?
git reset HEAD ~number

4.Как создать новую ветку и переключится в нее?
git checkout -b "branch name"

5.В чем отличение между fast-forward merge  and non fast-forfard merge?
В случае fast-forward merge текущая ветка не имеет дополнительных коммитов 
в отличии от ветки, которую мы merge'ым.


6.Когда может возникнуть конфликт слияния веток, как  его разрешить?
Когда две ветки, которые мы пытаемся смерджить, имеют изменения в одной строке в одном и том же файле,
или несовпадающие изменения.
В этом случае Git попросит вас помочь решить, какой из двух вариантов мы хотим сохранить.
При попытке объединить ветки, Git покажет, где происходит конфликт. 

7.Можно ли удалить ветку? Как? 
git branch -d "branch name"
